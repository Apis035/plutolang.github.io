## base32
### base32.encode(data, pad)
Returns an encoded base32 string.
#### Parameters:
1. `data` — A string of data to encode.
2. `pad` — Whether or not to apply padding.
### base32.decode(data)
Returns a decoded base32 string.
#### Parameters:
1. `data` — A string of base32 data to decode.
